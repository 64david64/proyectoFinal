Descripción del Proyecto y Tecnologías

Motivación del Proyecto

Este proyecto nace como respuesta a una problemática tangible y cotidiana: las dificultades en el transporte público que afectan directamente la calidad de vida de miles de personas en Soacha. El objetivo principal es investigar y proponer visualizaciones que ayuden a comprender mejor la situación, con la esperanza de que esta información pueda contribuir a mitigar el impacto negativo en quienes dependen diariamente de este servicio esencial. Buscamos, de alguna manera, reducir los tiempos de espera, la congestión y mejorar la experiencia general del usuario.

Tecnologías Utilizadas

Para el desarrollo de este proyecto, se están empleando y se planea utilizar un conjunto de tecnologías web modernas y bibliotecas especializadas en el manejo de datos geoespaciales. La elección de estas herramientas se basa en su robustez, flexibilidad y la amplia comunidad de soporte que las respalda.

  • HTML (HyperText Markup Language): Es el esqueleto de todas las páginas web del proyecto. Se utiliza para estructurar el contenido de manera semántica, asegurando que la información sea accesible y bien organizada.
  • CSS (Cascading Style Sheets): Se encarga de toda la presentación visual y el diseño. Mediante `estilos.css` (para estilos generales) y hojas de estilo específicas por página (como `descripcion-estilos.css`), se busca ofrecer una experiencia de usuario agradable, intuitiva y coherente a lo largo del sitio.
  • JavaScript: Aporta la interactividad y dinamismo a las páginas. Es fundamental para manipular el DOM, gestionar eventos y, crucialmente, para interactuar con bibliotecas de mapeo y visualización de datos.
  • Leaflet.js: Una biblioteca de JavaScript de código abierto, ligera pero potente, para crear mapas interactivos amigables para móviles. Ha sido la elección para la visualización de mapas base y la representación de polígonos geográficos, como se ve en la página principal y la sección "Acerca de".
  • Turf.js (Próximamente): Se planea integrar Turf.js, una biblioteca avanzada de JavaScript para análisis geoespacial. Con Turf.js, podremos realizar operaciones como cálculos de áreas, intersecciones, buffers, y otras funciones espaciales directamente en el navegador. Esto será clave para el "componente práctico" del proyecto, permitiendo análisis más profundos y visualizaciones de datos espaciales más ricas y significativas.

La combinación de estas tecnologías nos permite construir una aplicación web que no solo presenta información, sino que también ofrece herramientas para el análisis y la visualización interactiva de la problemática del transporte en Soacha y a nivel general.
