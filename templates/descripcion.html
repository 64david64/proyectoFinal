Descripción del Proyecto y Tecnologías

Motivación del Proyecto

Este proyecto nace como respuesta a una problemática tangible y cotidiana: las dificultades en el transporte público que afectan directamente la calidad de vida de miles de personas en Soacha. El objetivo principal es investigar y proponer visualizaciones que ayuden a comprender mejor la situación.

Tecnologías Utilizadas

Para el desarrollo de este proyecto se emplean un conjunto de tecnologías web modernas, bibliotecas de Python y herramientas de desarrollo que permiten un flujo de trabajo eficiente y un producto final robusto.

Python

El lenguaje de programación principal para toda la lógica del servidor (backend) y el procesamiento de datos.

Flask

Un micro-framework de Python que utilizamos para construir la aplicación web, gestionar las rutas y servir las páginas.

SQLAlchemy

Una librería (ORM) que facilita la comunicación entre nuestra aplicación Python y la base de datos de una forma orientada a objetos.

HTML5

El lenguaje de marcado estándar para crear la estructura y el esqueleto de todas las páginas de nuestro portal.

CSS3

Se encarga de toda la presentación visual, el diseño, los colores y las animaciones que dan vida a la interfaz.

JavaScript

Aporta la interactividad y el dinamismo en el navegador del usuario, especialmente en la manipulación de los mapas.

Leaflet.js

La biblioteca de JavaScript especializada que nos permite crear y manipular los mapas interactivos del proyecto.

GitHub

Plataforma que usamos para el control de versiones (con Git) y para el despliegue automático mediante GitHub Actions.
